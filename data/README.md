# Datasets

This folder contains the three hotel booking datasets used in the project. All three come from publicly available hotel booking data on Kaggle.

| File | Records | Features | Description |
|------|---------|----------|-------------|
| `dataset_1_indian_resort_hotels.csv` | 119,390 | 33 | Resort hotel bookings labeled with Indian cities. An adapted version of the widely used Hotel Booking Demand dataset. |
| `dataset_2_inn_hotel.csv` | 36,275 | 19 | Inn hotel chain reservations with a binary `booking_status` target. |
| `dataset_3_inn_hotel_variant.csv` | 36,285 | 17 | A variant of the Inn hotel data with renamed columns. |

## Sources

- **Dataset 1** is based on the *Hotel Booking Demand* dataset: https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand
- **Dataset 2** is the *Hotel Reservations Dataset* by Ahsan Raza: https://www.kaggle.com/datasets/ahsan81/hotel-reservations-classification-dataset
- **Dataset 3** is a renamed variant of the same Inn hotel reservation data, redistributed on Kaggle.

All datasets are used here for educational and research purposes. Credit for the original data belongs to the respective Kaggle dataset authors.
